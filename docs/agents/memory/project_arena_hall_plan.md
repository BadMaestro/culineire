**Утверждено:**
1. Переезд аватаров (телепорты по лайфциклу битвы)
2. Зрительский попап ВСТРОЕН на арене (вариант A, не ссылка)
3. Battle Room страница = «прихожая» (правила/рейтинги/статы → переход на арену)

**Ключевая механика:** синие ячейки = зарегистрированные, серые поля = анонимы (СОЗДАТЬ — одобренное расширение геометрии); попап шефа по клику (статы + примерный атак/защит потенциал по артефактам, сами артефакты НЕ показывать + Challenge прямо в попапе); вызов принят → телепорт в пару друг напротив друга (одинаковый ранг = своё кольцо, разные ранги = вертикальная пара между кольцами); время битвы → телепорт в центр (2 ячейки + VS, переезд БЕЗ дублирования); вся центр-ячейка = одна ссылка на Battle Room popup (шеф слева/справа, артефакты видны — битва открытая, чат только этого боя, голосование, подарки); после боя возврат в исходные ячейки. Шефы ОБЯЗАНЫ использовать подаренные зрителями артефакты; Appreciation-подарки на бой не влияют и после боя могут быть «проданы».

**6 открытых вопросов владельцу (спросить перед стадией):** (1) синие ячейки — все залогиненные или только с кошельком; (2) анонимы — реальный подсчёт или символика; (3) формат «примерного потенциала»; (4) откуда шефы делают боевые ходы — попап или прихожая; (5) курс «продажи» appreciation-подарков + юр. проверка против closed-loop s14/s17; (6) кто назначает время битвы.
