Перед тем как говорить "это не сделано" или начинать писать код — проверить ВСЁ:
1. Локальный git log / файлы
2. Сервер (git log на проде, find по файлам)
3. CoWork журнал (`CoworkingLogEntry` — что GreenBear уже сделал)

**Why:** Сказал пользователю "ничего нет" основываясь только на локальном репозитории, хотя Фаза 1 Arena уже была задеплоена на сервере GreenBear'ом.

**How to apply:** При любом "read only / проверь что уже есть" — обязательно проверить сервер и CoWork перед любыми выводами.

**Дополнение 2026-07-05 (UI):** Перед добавлением UI-элемента ("нет кнопки X") — сначала проверить ВЖИВУЮ в браузере, не рендерится ли эквивалент уже где-то (включая detached/absolute-позиционированные попапы, которые визуально не видны на месте, но существуют в DOM). Добавил дубль Sign Out в drawer, не заметив, что оригинал из попапа профиля рендерился отдельно ниже drawer-а. Проверка: querySelectorAll по всем кандидатам + getBoundingClientRect (height>0, координаты) в обоих состояниях (свёрнуто/раскрыто).
