**Ссылка для показа/сравнения арены (over-in-development):**

    https://culineire.ie/chef-battle/arena/?proto=1

`?proto=1` — **прото-гейт**: та же прод-ссылка, параметр переключает на НОВУЮ
процедурную арену (SVG, полярная геометрия). Обычные посетители без параметра её
не видят → арену пилят прямо на проде за этим флагом, **отдельного staging-домена
нет**. (Гейт ?proto=1 — работа Ember, влит ~v2.5.304.)

**НЕ давать вместо этого:** голый `/chef-battle/arena/` (это старая/дефолтная
арена), dev-сервер на localhost, или «прод вообще». Владелец бил за это 2026-07-18:
«мы не пишем арену на живой прод… для этого есть временная другая ссылка» — имелась
в виду ИМЕННО ?proto=1.

**Ограничение:** арена под dark launch (chef_battle_guard → 404 для не-battle-visible).
Залогиненный GreenBear/суперюзер видит; аноним (в т.ч. мой in-app браузер без
авторизации) получит 404. Значит пиксель-в-пиксель сравнение с мокапом я сам снять
НЕ могу — прошу у владельца скриншот ?proto=1, как он прислал мокап.
