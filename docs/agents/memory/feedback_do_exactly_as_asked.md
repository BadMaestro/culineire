Когда задание конкретное ("скопируй X на все страницы Y") — делать именно это, не изобретать JS-инжекции, context processors или другие "умные" решения.

**Why:** Пользователь разозлился когда вместо прямого копирования panel HTML в шаблоны был написан JS injection через <template> тег — это сломало визуал, добавило сложности и потребовало 3 итерации исправлений.

**How to apply:** Если пользователь говорит "скопируй/добавь X туда-то" — делать буквально: копировать, вставлять include, не придумывать альтернативную архитектуру.
