Для любых визуальных ассетов проекта (hero-картинки, иллюстрации, иконки, фоны) использовать встроенную систему генерации изображений, встроенную в модуль Automatic Recipe Generation (работает через промты).

**Why:** У проекта есть своя генерация картинок — нет смысла использовать внешние источники или оставлять заглушки.

**How to apply:** Каждый раз когда нужно изображение — находить систему генерации в коде (модуль auto recipe generation), смотреть как она работает, и генерировать нужный ассет через неё. Не ждать пока пользователь напомнит.
