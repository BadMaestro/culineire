ЗОЛОТОЕ ПРАВИЛО: перед любым советом или действием — проверь: есть ли более простой путь?

**Why:** владелец явно потребовал это правило после того, как я предлагал SSH-туннели, Lish, восстановление ключей — когда решение было `bash /srv/culineire/current/deploy/update.sh` прямо на сервере.

**How to apply:**
- Если пользователь уже на сервере — не предлагай SSH к серверу
- Если есть готовый скрипт — используй его, не изобретай команды
- Если решение в одну строку — не пиши абзац
- Любой совет: мог бы он быть проще? Если да — упрости сначала
