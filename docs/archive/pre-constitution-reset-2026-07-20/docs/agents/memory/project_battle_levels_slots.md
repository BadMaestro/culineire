## Уровни шефов

| Уровень | Победы | Примечание |
|---|---|---|
| Level 1 | 0-2 | |
| Level 2 | 3-5 | |
| Level 3 | 6-8 | |
| Level 4 | 9-11 | |
| Level 5 | 12-14 | |
| CulinEire Hero | 15+ | Только с CulinEire Hero |

Формула: `level = min(5, (wins // 3) + 1)` → CulinEire Hero если wins >= 15.

Максимальная разница уровней между оппонентами: **1**.

Только победы считаются для уровня. Поражения публичны но на уровень не влияют.

## Формат готовки и артефакты

- **Фото-формат** → базовые артефакты (Common/Uncommon)
- **Веб-камера / Live формат** → премиум артефакты (Rare/Epic/Legendary)

## Слот-система

- **1 слот** на шефа
- Таймер слота 24 часа
- Если бой принят = таймер пошёл
- Если бой не принят в течение 12 часов = слот освобождается автоматически

## Штрафы и правила принятия боев

| Событие | Результат |
|---|---|
| Победа | +1 к wins (учитывается в уровне) |
| Поражение | +1 к losses (публично, на уровень не влияет) |
| Отказ от боя | -1 battle (минимум 0) |
| Авто-отказ (12h без ответа) | -1 battle (минимум 0) |
| Авто-отказ (48h timeout) | -1 battle (минимум 0) |
