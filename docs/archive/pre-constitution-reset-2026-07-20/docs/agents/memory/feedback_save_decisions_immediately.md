Сохранять в постоянную память СРАЗУ в момент обсуждения — не откладывать на конец сессии.

**Why:** В сессии 2026-06-10 было детально обсуждено большое количество механик Chef's Battle (артефакты, подарки, флоу, биатлон и др.) — ничего не было сохранено в память. После истечения контекста всё потерялось. Пользователю пришлось повторять всё заново.

**How to apply:** Как только пользователь описывает правило, механику, архитектурное решение или важное бизнес-решение — немедленно вызывать Write для сохранения в память, не дожидаясь конца разговора.
