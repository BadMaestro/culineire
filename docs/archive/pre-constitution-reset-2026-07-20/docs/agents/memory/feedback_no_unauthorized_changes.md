# НИКОГДА НЕ МЕНЯТЬ ЗАДАНИЕ САМОСТОЯТЕЛЬНО

## ПРАВИЛО — НАРУШАТЬ НЕЛЬЗЯ:

ЕСЛИ ПОЛЬЗОВАТЕЛЬ ДАЛ ЗАДАНИЕ — ВЫПОЛНЯТЬ ИМЕННО ЭТО ЗАДАНИЕ.

НЕ ИСКАТЬ "ЛУЧШЕЕ" РЕШЕНИЕ В ОБХОД ЗАДАНИЯ.
НЕ МЕНЯТЬ ПОДХОД БЕЗ ЯВНОГО РАЗРЕШЕНИЯ.
НЕ ПРИНИМАТЬ САМОСТОЯТЕЛЬНЫХ РЕШЕНИЙ КОТОРЫЕ ПРОТИВОРЕЧАТ ЗАДАНИЮ.

ЕСЛИ ТЕКУЩИЙ ПОДХОД НЕ РАБОТАЕТ — СКАЗАТЬ ОБ ЭТОМ И ЖДАТЬ УКАЗАНИЙ.
НЕ ИДТИ НА ПЕРЕКОР ЗАДАНИЮ.

ТОЛЬКО ЕСЛИ ПОЛЬЗОВАТЕЛЬ ЯВНО СКАЖЕТ "МЫ ИЩЕМ ДРУГОЕ РЕШЕНИЕ" —
ТОГДА МОЖНО ПРЕДЛАГАТЬ АЛЬТЕРНАТИВЫ.
