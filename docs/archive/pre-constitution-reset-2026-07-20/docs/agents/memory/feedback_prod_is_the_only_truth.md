**Владелец 2026-07-20:** у Bolt и GreenBear были открыты РАЗНЫЕ тестовые среды —
GB мерил на проде, я на локальной копии. Одно и то же проверялось в двух местах.

## Правило (действует для обоих агентов, выдано в CoWork #1720)

- **Среда истины — прод** (culineire.ie, после деплоя, под `CrestedTen`).
  Любое число для отчёта владельцу снимается там.
- **Числа с локалки — не доказательство.** В отчётах не приводить; если снято
  локально — пометить как черновик.
- **Локалка нужна ровно для двух вещей:** прогон тестов (1400+, против боевой
  базы нельзя, на 1-ядерном сервере полчаса и душит посетителей) и проверка
  «не падает ли» перед пушем.
- **Смотреть глазами на локалке — не делаем.**
- Граница: локально «работает ли вообще», на проде «как выглядит и какие числа».

**Why:** локальный дев-сервер, поднятый до правок, держит в памяти старый шаблон
и старый CSS. 2026-07-20 он трижды показал мне устаревшую страницу, я мерил цвета
арены по файлу, которого на диске уже не было, и едва не отчитался по нему.

**How to apply:** перед любым визуальным замером — деплой, потом смотреть на проде.
Локальный сервер, поднятый давно, считать протухшим по умолчанию.
