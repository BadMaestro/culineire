Do not use em dashes (—) or double dashes (--) in user-facing text in templates, hints, labels, messages.

**Why:** User preference — dashes look wrong in the site's tone.

**How to apply:** Use a period, comma, or rewrite the sentence instead. OK in git commit messages and code comments, not in HTML templates.
