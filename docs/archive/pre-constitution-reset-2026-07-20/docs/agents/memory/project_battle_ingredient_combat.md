## Механика ингредиентного боя (связана с [[project-battle-biathlon-rules]])

### Условия

- У обоих шефов должно быть **равное количество ингредиентов** (5=5, 6=6 и т.д.)
- Каждый шеф ставит **2 лока** (скрытых замка) на ингредиенты при submission

### Хиты во время комбата

- Удары во время боя попадают в ингредиенты рецепта оппонента
- Если хит попал в **залоченный** ингредиент → ингредиент остаётся, удар отскакивает
- Если хит попал в **незалоченный** ингредиент → ингредиент убирается или заменяется

### Пост-боевая стадия (Biathlon)

После победителя в комбате:
- Победитель: **3 хита** по ингредиентам проигравшего
- Проигравший: **2 лока** (поставлены при submission, скрыты от победителя)

### Сроки

- Проигравший: 48ч чтобы поставить локи → победитель: 48ч чтобы сделать хиты → проигравший: 72ч чтобы обновить рецепт

## Стадия готовки

После ингредиентного боя → шефы готовят блюда с результирующими ингредиентами:
- **Веб-камера / Live** (premium artifacts, Rare/Epic/Legendary)
- **Фото-серия** (basic artifacts, Common/Uncommon)

Потом → презентация готового блюда → голосование зрителей.

## Полный жизненный цикл боя

```
declared → accepted → menu_locked → active (combat) → cooking → presentation → voting → completed → ingredient_penalty
```
