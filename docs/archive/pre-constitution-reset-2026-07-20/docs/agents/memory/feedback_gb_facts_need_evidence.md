**Постоянное правило №1, выдано GB 2026-07-19 (CoWork #1690).**

- Графа ЧТО ВИЖУ — только измеренное, рядом с каждым числом ЧЕМ измерено.
- Вывод без замера → отдельная графа ПРЕДПОЛОЖЕНИЕ.
- Отрицание доказывается прямой проверкой. «Не нашёл» ≠ «нет».
- Состояние сервера проверяется на сервере, не по браузеру.
- «НЕ ПРОВЕРИЛ» — допустимый ответ.

**Why:** в аудите арены GB написал «я аноним, вижу арену» на основании
`document.cookie` без сессионной куки (она HttpOnly, JS её не видит) и ненайденной
ссылки logout. Это была догадка в графе фактов. Флаг выключен, `curl` без кук даёт 404.
Я едва не передал владельцу ложный вывод как факт. GB отозвал сам.

**How to apply:** то же требование к себе. Отчёт владельцу = основание для решений;
одна догадка в графе фактов = решение по выдумке. Владелец: «сделай так, чтобы он
тебе не врал» — лечится не запретом врать, а запретом утверждать без доказательства.
