Всё новое что создаётся — сразу максимально синхронизировать со стилями проекта: те же шрифты, те же кнопки (CSS-классы btn, btn--primary и т.д.), те же цвета (CSS-переменные --brand, --ink, --surface и т.д.), тот же дизайн-язык.

**Если стоит вопрос выбора стиля** — СНАЧАЛА читать `static/css/base.css` (`:root` дизайн-токены), и только потом писать CSS. Не угадывать цвета по памяти. Основная палитра: `--bg: #faf6f0`, `--brand: #184c3a`, `--brand-mid: #8b7355`, `--border: #e2d9cc`, `--surface: #fffdf9` — тёплые кремово-пергаментные тона, не зелёные.

**Why:** Был случай — написал зелёный glassmorphism для виджета, хотя сайт использует тёплые кремовые тона. Переделывал после замечания.

**How to apply:** Перед написанием нового шаблона или CSS — открыть base.css и посмотреть существующие переменные и компоненты. Использовать их, не изобретать новые классы без необходимости. При любом сомнении — grep `:root` в base.css.
