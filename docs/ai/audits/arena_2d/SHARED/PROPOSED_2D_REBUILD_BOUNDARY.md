# Proposed 2D Rebuild Boundary

## Inside

- Public Arena semantic 2D composition.
- One scoped token-compliant stylesheet.
- Native controls replacing SVG polygons.
- Adapted phase/metrics/chef/crown/gift/empty components.
- Approved popup or canonical-room navigation.
- Browser/responsive/keyboard/screen-reader/reduced-motion tests.
- AMC compatibility or decoupling before renderer retirement.

## Outside

- Models, migrations, services, scoring, challenges and lifecycle.
- Access, participant, moderation and operator permissions.
- Submission secrecy, voting integrity and fraud evidence.
- Token/gift/artifact/reward/ledger transactions.
- Named URLs/actions and broadcast snapshot.

## Safe sequence

1. Add context/poll/action contract tests.
2. Decide popup and operator-preview scope.
3. Build 2D against existing contracts.
4. Prove AMC compatibility.
5. Verify browser/action parity.
6. Isolate geometry/cinematic presentation.
7. Separate dead-code audit before deletion.

This is a boundary proposal, not implementation authority.
